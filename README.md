## üß© Project Overview  

The **Automotive Object Detection** project focuses on applying Artificial Intelligence to identify and analyze vehicles in real-time using **YOLOv8** (You Only Look Once version 8).  
The goal is to create an intelligent system capable of detecting cars, buses, and trucks from road or aerial traffic videos to estimate **traffic density** and assist in **smart city traffic management**.  

This project supports sustainable mobility by helping optimize traffic flow and reduce fuel wastage, contributing to **green technology development**.  

---

## üß† Problem Statement  

In modern cities, increasing vehicle congestion leads to:  
- Long travel times and fuel consumption  
- Traffic mismanagement  
- Air pollution due to idling vehicles  

A system that can automatically **detect and count vehicles in real-time** can help traffic authorities make data-driven decisions.  
Thus, this project aims to **detect, classify, and analyze vehicles using AI** for improved road traffic management and sustainability.
